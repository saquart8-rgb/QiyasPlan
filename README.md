# QiyasPlan: AI-Driven Educational Platform 

**QiyasPlan** is a high-performance educational platform designed to help students prepare for standardized tests (Qiyas) in Saudi Arabia. The platform leverages AI-driven content generation and scalable cloud infrastructure to provide a seamless learning experience.

##  Impact & Metrics
- **7,800+ Registered Users** within the first 14 days of launch.
- **800+ Daily Active Users (DAU)** with stable performance.
- **5,000+ Monthly Active Users (MAU)**.
- **90%+ Google Lighthouse Score** in performance, accessibility, and SEO.

##  Tech Stack
- **Frontend:** Next.js / React.js (Optimized for speed and SEO).
- **Backend & Database:** Firebase (Firestore, Authentication, and Cloud Functions).
- **AI Integration:** Custom-engineered prompts for educational content generation and automated testing materials.
- **Deployment:** Vercel / Firebase Hosting with automated CI/CD pipelines.

##  Key Achievements
- **Scalability:** Engineered the system to handle a massive influx of users (7K+) in a very short period without downtime.
- **Technical Problem Solving:** Successfully managed international support communication to upgrade cloud quotas and bypass regional restrictions within 48 hours.
- **User Experience:** Achieved top-tier web vitals, ensuring a fast and responsive interface for students.

##  Source Code
To protect the business logic and user data privacy, the core source code is maintained in a private repository. This repository showcases the project's impact, technical architecture, and successful market launch.
